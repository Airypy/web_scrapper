Tech Used:-
This is a webscrapper made using beautifulsoup4 and selenium with webdriver.

What it Does:-
1.artoflivingQNA-Run the single file inside the folder and put the url whose QNA you want to scrap,remember it will scrap the entire QNA till the end page.

2.product_scrapper:-Run the product_scrapper.py,It will the url of the page whose product details you want to scrap.
Then It saves the scrapped data in the CSV files one at a runtime.
